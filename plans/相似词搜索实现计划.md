# 相似词搜索功能实现计划

## 概述
在单词无法拆分前后缀时，使用整个单词作为词干进行相似词搜索和打分。

## 功能模块

### 1. 相似词搜索模块 (`app/related_words.py`)

#### 1.1 核心类: `RelatedWordsSearcher`

**初始化:**
- 加载 COCA60000 词频数据（复用 `WordFreq` 类）
- 加载前缀列表 (`数据资料/_prefixes.txt`)
- 加载后缀列表 (`数据资料/_suffixes.txt`)
- 加载常见字母组合 (`数据资料/常见字母组合.json`)

**方法:**

1. `search_and_score(stem: str, max_results: int = 50)` 
   - 输入词干，返回打分排序的相似词列表
   - 使用 `*stem*` 正则搜索所有匹配词
   - 对每个匹配词计算相似度分数
   - 按分数降序、词频升序排序
   - 返回前 max_results 个结果

2. `calculate_score(stem: str, word: str) -> int`
   - 计算词干与单词的相似度分数
   - 分数越高越相似
   
   **打分规则:**
   
   核心逻辑：找出词干与单词之间的"差异部分"，对差异部分逐段打分。
   
   步骤1: 找出差异部分
   - 如果 word 包含 stem 作为子串：
     - 检查前缀差异（word开头在stem之前的部分）
     - 检查后缀差异（word结尾在stem之后的部分）
   - 如果 stem 包含 word 作为子串：同理
   - 如果有嵌入差异（stem在word中间）：前后都有差异
   - 如果没有包含关系：使用编辑距离
   
   步骤2: 对差异部分打分
   - 每个**单独的字母**算1分
   - 如果发现一个完整的**前缀/后缀/常见字母组合**，整个组合只算1分（不管多长）
   
   示例:
   - `ation` 是已知后缀 → 算1分
   - `pre` 是已知前缀 → 算1分
   - `bl` 是常见组合 → 算1分
   - 单个字母 `x` → 算1分
   - `pre` + `x` + `ation` → 1+1+1 = 3分

3. `_get_diff_score(stem: str, word: str) -> int`
   - 计算两个字符串的差异分数
   - 如果 word 包含 stem 作为子串：
     - 前缀差异：word 开头多出的字母数
     - 后缀差异：word 结尾多出的字母数
     - 总分 = 前缀差异 + 后缀差异
   - 如果 stem 包含 word 作为子串：同理
   - 否则：使用编辑距离

4. `_check_prefix_bonus(extra: str) -> bool`
   - 检查额外部分是否是已知前缀

5. `_check_suffix_bonus(extra: str) -> bool`
   - 检查额外部分是否是已知后缀

6. `_check_common_combo_bonus(text: str) -> bool`
   - 检查文本中是否包含常见字母组合

### 2. 增强模块 (`app/enhanced_words.py`)

**触发条件:** 相似词搜索结果少于 15 个时触发增强

**方法:**

1. `enhance_with_vowel_replace(stem: str, cocawords: set) -> list[str]`
   - 替换词干中的元音，生成新词干
   - 检查新词干是否在 COCA60000 中
   - 返回找到的合法单词

2. `enhance_with_consonant_replace(stem: str, cocawords: set) -> list[str]`
   - 替换词干中的辅音，生成新词干
   - 检查新词干是否在 COCA60000 中

3. `enhance_with_affix_addition(base_words: list[str], cocawords: set) -> list[str]`
   - 给已有单词添加前后缀
   - 检查是否在 COCA60000 中

- 举例：`wait` 中有双元音 `ai` (/eɪ/)，可以替换为 ay, a, ey 等

## 第一步详细设计：打分算法

### 场景1：词干是单词的子串
- `situ` vs `situation` → situation 包含 situ
- 后缀差异: `ation` (4个字母)
- 检查 `ation` 是否是已知后缀 → 是，+1分
- 总分 = 4 + 1 = 5分

### 场景2：词干与单词有部分重叠
- `sit` vs `sits` → sits 包含 sit
- 后缀差异: `s` (1个字母)
- 总分 = 1分

### 场景3：词干与单词完全不同
- 使用编辑距离计算差异
