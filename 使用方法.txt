# 怎样运行
已成功创建基于 FastAPI + htmx 的 TLD 词典查询网站，核心功能已完成：

1. **后端** - FastAPI 服务器运行在 `http://localhost:8000`
   - `MDXReader` 类：启动时加载 TLD.mdx 的 4,362,467 条词条索引（~30秒），查询时二分查找 + 按需 zlib 解压记录块，性能优秀
   - `MDDReader` 类：全量缓存 149,213 个多媒体资源到内存，查图片/音频秒回
   - `/api/lookup?word=xxx` 精确查词 + 前缀拼写建议
   - `/api/media/{path}` 代理多媒体资源（图片、音频）
   - `/static/p.css` 提供词典原生样式
   - `/static/style.css` 提供页面布局样式

2. **前端** - htmx 驱动的单页应用
   - 漂亮的渐变头部 + 居中搜索框
   - 回车搜索 / 点击按钮搜索
   - 输入加载动画
   - 自动替换词典资源路径代理到后端
   - 完整词典样式展示（音标、发音图标、词频、等级标注等）

3. **目录结构**:
   - `app/main.py` — 主程序
   - `app/test_dict.py` — 测试脚本
   - `templates/index.html` — 首页 HTML 模板
   - `static/style.css` — 页面样式
   - `The little dict/` — 词典数据（被 gitignore 排除）

4. **启动命令**: `cd web2mdd && python -m app.main`
   或: `cd web2mdd && venv\Scripts\python -m app.main`
