### 1. 获得词频的值

**状态**: ✅ 已实现

**API**: `GET /api/rank?word=splice`

从词典结果中提取第一个 RANK 标签后面的数字作为词频值（取所有 RANK 中的最小值），通过独立 API 接口返回 JSON。

```json
{"word": "splice", "rank": 14588, "found": true}
```

如果单词存在但没有词频字段，返回 `rank: null`；如果单词不存在，返回 `found: false`。

### 3. 能找出单词的词根词缀，同源词

**状态**: 🔶 部分已实现

#### 3a. 词根词缀切分 ✅ 已实现
